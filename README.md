# Microsoft-Cognitive-Service-OCR-Demo-.NET-MVC

this repository contaians a Asp.net MVC 5 solution website.

When you uploaded an image file,  the TextHelper.cs in this solution,  will send a request to Microsoft Cognitive Service OCR API, OCR API detects text in the uploaded image and extracts the recognized characters, once receive response message from OCR API,  then convert response data to a object, at last display text on the page

Learn more about Microsoft Cognitive Service:

https://azure.microsoft.com/zh-tw/services/cognitive-services/
https://channel9.msdn.com/Series/cognitive/7mins-overview
